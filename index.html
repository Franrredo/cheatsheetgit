Cheatsheet GIT

Voy comenzar una pequeña guia recordatorio de Git para poder tenerla a mano algunos de estos comandos incluiran breves explicaciones.

Configuración

  • git config --global user.name "nombre apellidos"
  • git config --global user.email "demo@demo.es"
  • git config --list muestra todas las configuraciones
  • git help "valor" muestra la ayuda respecto a un termino
